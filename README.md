## Tech Stack
**Client:** React, Redux, Chakra UI, CSS Modules, Styled-Components,BootStrap
**Server:** Node,FireBase
## Run Locally
Clone the project
```bash
  git clone https://github.com/Shravankumar1234/Farfetch_Project
```
Go to the project directory
```bash
  cd Farfetch-clone
```
Install dependencies
```bash
  npm install
```
Start the server
```bash
  npm run start
```
